
# Activity2_POO

## Description

Activity2_POO is a project developed as part of an educational program to demonstrate and apply the principles of Object-Oriented Programming (OOP). This project aims to provide a practical understanding of OOP concepts such as classes, objects, inheritance, polymorphism, encapsulation, and abstraction. It is implemented in [Programming Language], making it a valuable resource for students and educators alike who are exploring the fundamentals of OOP.

## Features

- Implementation of basic OOP concepts.
- Examples of classes and objects.
- Demonstrations of inheritance and polymorphism.
- Usage of encapsulation and abstraction principles.
